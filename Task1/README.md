# Task 1 – Mobile App Signup Flow

## CODSOFT UI/UX Internship

### Objective

The objective of this task is to design a simple, user-friendly, secure, and visually appealing mobile app signup flow.

The signup flow helps users create an account quickly while collecting only the necessary information.

---

## Project Description

This project is a mobile app signup flow designed for a food delivery application.

The design focuses on:

* Simple and clean UI
* Easy navigation
* Secure signup process
* Personalized onboarding experience
* Mobile-friendly layout

---

## Screens Included

1. Welcome Screen
2. Signup Screen
3. OTP Verification Screen
4. Personalization Screen
5. Success Screen
